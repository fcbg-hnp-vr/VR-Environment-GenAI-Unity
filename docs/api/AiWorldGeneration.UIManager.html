    Declaration
    public EditionMode GetEditionMode(int value)
    Parameters
    Type Name Description
    int value

    The index of the selected option in the edition mode dropdown.
    Returns
    Type Description
    EditionMode

    The corresponding edition mode based on the selected dropdown value. If the selected option is "skybox creation", returns EditionMode.BASIC. If the selected option is "skybox edition", returns EditionMode.PAINTING. If the selected option is "expert mode", returns EditionMode.EXPERT. Throws an InvalidOperationException if the selected dropdown value does not match any of the expected options.
